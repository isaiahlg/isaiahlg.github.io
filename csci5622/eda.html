Exploratory Data Analysis

Demographic & Health Surveys

A key source of survey data from sub-Saharan Africa and other developing countries comes from the Demographic and Health Surveys (DHS) Program from the United States Agency for International Development (USAID).

Raw data from DHS must first be requested through the information request form, with the intended use of the data registered. Once access is granted, the data can be fetched in two ways: (1) by downloading a zip file folder from the website, and (2) by requesting data over the API. The documentation for the API is limited but can be found on the DHS website here. A sample of a downloaded zip file can be found here. To use the API, an R package was developed by a group of researchers for the rOpenSci project and can be found on GitHub. The implemntation of this code for pulling data from the API, processing it, and cleaning the data can be found on GitHub
